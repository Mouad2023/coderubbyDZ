Code Rubby - User Guide

Welcome to Code Rubby

Your AI-powered code generator using DeepSeek via Novita AI!

  • Enter a Prompt: On the next page, type a coding request (e.g., "Create a login API in Flask") in the textarea.
  • Submit: Click "Generate" to get a response from DeepSeek.
  • View Results: The response appears below the form, with code formatted for clarity.
  • History: Past prompts and responses are shown in the "History" section, saved in your browser.
  • Theme Toggle: Switch between light and dark themes using the button above.
  • Clear History: Use the "Clear History" button to reset saved responses.
